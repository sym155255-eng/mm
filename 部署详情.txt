============================================================
 导航站 —— 用「部署包」上传部署详情
 （配合文件：导航站-部署包.tar.gz）
============================================================

适用：拿到一台新服务器（Ubuntu 22.04），用本地打的包部署。
（如果想改用 git 拉取代码部署，见另一份「换服务器部署步骤.txt」）

------------------------------------------------------------
 第 ① 步：装环境（Node + git + pm2）
------------------------------------------------------------
SSH 登录服务器（root），执行：
  curl -fsSL https://deb.nodesource.com/setup_20.x | bash - && apt install -y nodejs git && npm install -g pm2
验证：node -v   （出现 v20.x 即可）

------------------------------------------------------------
 第 ② 步：上传部署包
------------------------------------------------------------
 装好 1Panel 后用「文件」管理上传，或用 scp：
  - 把本地 导航站-部署包.tar.gz 上传到服务器 /root 目录

------------------------------------------------------------
 第 ③ 步：解压成 ~/ddd
------------------------------------------------------------
  mkdir -p ~/ddd && tar -xzf ~/导航站-部署包.tar.gz -C ~/ddd && ls ~/ddd
 看到 client / server / data 等目录即成功。

------------------------------------------------------------
 第 ④ 步：构建前端 + 装后端 + 启动
------------------------------------------------------------
  cd ~/ddd/client && npm install && npm run build
  cd ~/ddd/server && npm install --omit=dev
  pm2 start src/index.js --name nav-site
  pm2 save
  pm2 startup systemd -u root --hp /root      # 按提示再粘贴它输出的那条命令

 验证：curl -s http://127.0.0.1:3001/api/public/data | head -c 100
       （返回一串 JSON 即正常）

------------------------------------------------------------
 第 ⑤ 步（可选）：开启自动备份
------------------------------------------------------------
  printf '%s\n' 'BACKUP_REPO=https://你的token@github.com/aa1558686/nav-db-backup.git' 'BACKUP_INTERVAL_MIN=60' > ~/ddd/server/.env
  pm2 restart nav-site
 （"你的token" 换成 github_pat_ 那段，@github.com 不动）

------------------------------------------------------------
 第 ⑥ 步：域名反代 + HTTPS（1Panel 面板）
------------------------------------------------------------
 1) 装 1Panel：
    curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && bash quick_start.sh
 2) 网站 → 创建 → 反向代理：
      域名 1558686.xyz   代理地址 http://127.0.0.1:3001
 3) 证书 → Acme账户(邮箱) → 申请证书(域名 1558686.xyz，验证方式 HTTP)
 4) 网站 → HTTPS → 启用 + 选证书 + 强制 HTTPS
 （前提：域名 DNS 解析到该服务器 IP）

------------------------------------------------------------
 默认账号
------------------------------------------------------------
  后台：https://域名/admin   账号 admin   密码 admin123
  （登录后尽快在「网站设置」改密码）

------------------------------------------------------------
 日常更新（部署包是 git 仓库时才可用）
------------------------------------------------------------
 上传包部署的目录默认不是 git 仓库。若想用 update.sh 更新，
 先在 ~/ddd 里执行一次：
   git init
   git remote add origin https://github.com/aa1558686/sss.git
   git fetch origin
   git reset --hard origin/main
   git branch -M main
   git branch --set-upstream-to=origin/main main
 之后更新只需： cd ~/ddd && bash update.sh
